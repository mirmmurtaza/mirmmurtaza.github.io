Diving into the World of Data Science: A curated collection of projects and experiments

  • Facebook Link Prediction: Friend Recommendation using Machine Learning

    This is a Jupyter Notebook that contains the implementation of various Machine Learning algorithms to develop a Facebook Friend Recommendation System. It includes tasks such as data cleaning, manipulation, and feature engineering, in addition to the implementation of the model itself.

  • Quora Question Pair Similarity using Deep Learning

    This Jupyter notebook presents the implementation of Long Short-Term Memory (LSTM) based Deep Learning models for determining the similarity of question pairs on Quora. It also covers the preprocessing and conversion of text data into a suitable numerical representation for the model to work with.

  • Image Segmentation using U-NET and CANET

    This Jupyter Notebook demonstrates the training and application of U-NET and CANET models for the task of image segmentation on traffic images. The goal of this project is to aid in the development of self-driving cars.

  • Spoken Digit Recognition

    This project involves the categorization of sound data into various classes. The sound data pertains to spoken digits. The project was executed by constructing LSTM models, both on the raw input data as well as by converting the raw data into spectrograms.
